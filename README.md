# Enterprise-IT-Automation-Lab
This project demonstrates IT automation tasks commonly performed in enterprise
Windows environments using PowerShell and Active Directory.

All scripts were developed and tested in a controlled lab environment using
Windows Server 2025 and a domain-joined Windows client.

## Environment
- Windows Server 2025 (Domain Controller)
- Active Directory
- Windows 11 Client (Domain Joined)
- PowerShell 5+/7+

## Automations Included
- Active Directory user provisioning
- Endpoint system auditing
- Account hygiene and security reporting

## Skills Demonstrated
- Active Directory administration
- PowerShell automation
- Identity and access management
- Endpoint visibility
- Security best practices

## Disclaimer
All work was performed in a lab environment.
No production systems or data are included.
